Welcome to AXS’s documentation!¶

This is the main documentation site for AXS (Astronomy eXtensions for Spark).

The source code is hosted at two GitHub repositories: the repository containing AXS API and the repository containing Spark changes

To report a bug please send an email to one of the following addresses: pzecevic@uw.edu, ctslater@uw.edu, and mjuric@uw.edu
